jQuery rTable

jQuery plugin which allows you to make your table's columns resizable

View the Project on GitHub eplaksin/jquery-rtable

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Requirements

jQuery 1.3.2+

Installation

Download ZIP archive and unpack it to your project folder. Include CSS and JS files from /dist folder:

<link rel="stylesheet" href="dist/jquery.rtable.min.css">
<script src="jquery.min.js"></script>
<script src="dist/jquery.rtable.min.js"></script>
